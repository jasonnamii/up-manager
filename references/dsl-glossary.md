# DSL Glossary — KR↔EN 매핑 (v3.0 역호환)

> v3.0 정책상 KR 마스터. EN 표기는 v2.x 이전 호환·외부 문서 참조용.

## 축명

| KR (마스터) | EN (참고) |
|---|---|
| 진실성 | TRUTH |
| 독립성 | INDEPENDENCE |
| 통화성 | CURRENCY |
| 간결성 | BREVITY |

## 키워드

| KR (마스터) | EN (참고) |
|---|---|
| 정정 | CORRECTION |
| 미상 | UNKNOWN |
| 가정 | ASSUMPTION |
| 실패 / FAIL | FAIL |
| 최상위 실패 | TOP-LEVEL FAIL |
| 통화단위 위반 | FX_UNIT_CRIT |
| 규칙충돌 | CONFLICT_RULES |

## 룰 키

| KR (마스터) | EN (참고) |
|---|---|
| 트리거 / 발동조건 | TRIGGER |
| 규칙 | RULE |
| 점검 / 검사 | CHECK |
| 가드 / 차단 | GUARD |
| 비용 | COST |
| 범위 | SCOPE |
| 범위제외 | SCOPE_OUT |
| 범위포함 | SCOPE_IN |
| 변환 | CONVERT |
| 송출검사 / 자체검사 | SELF_CHECK |

## 보존 (영문 그대로)

- 기호: `::=` `★` `✗` `→` `∨` `∧` `①~④` `•`
- 정규식: `[0-9]+\s*(...)` 등 본문 그대로
- 업계 표준: BEP·KPI·MECE·MVP·API
- 고유명사·MCP 도구명·스킬명
