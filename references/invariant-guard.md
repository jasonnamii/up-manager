```

2차: 단일블록 헤더 (DSL EN 블록 0건 확인)
3차: 코드블록 4-backtick 래핑 + 기호 보존
```

## 위반 등급

| 등급 | 처리 |
|---|---|
| HIGH | 즉시 STOP + 보고 |
| MID | 자동 보정 1회 시도 후 보고 |
| LOW | 보정 + 인라인 경고 |

## 자동 보정 가능

- 코드블록 3-backtick → 4-backtick
- EN 헤더 잔존 → KR 단일화
- 키워드 영문 → KR (glossary 매핑)

## 자동 보정 불가 (STOP)

- 고유명사 로마자화 (의도성 판단 불가)
- 사용자 의도와 충돌 (호칭 변경 등)
- 2회차 실패 시
