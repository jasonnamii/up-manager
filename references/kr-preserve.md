## 유지 대상 ENUM

### 1. 트리거명 (26개 + 콤보 8개)

```
트리거 26개: 홈즈, 오컴, 제1원리, 베이지안, 모순기술, 엄브렐러, 아날로지,
  연역수렴, 외과적, 수정4, 백본, 스켈레톤, SHE, 엘베피치, 타임스톤,
  맥가이버, 넛지, 프리모르템, 트리아지, 줌, 절대자,
  부작업, 주작업, 제출청소, 이쁘니, 작업설계자

콤보 8개: trigger-dictionary SKILL.md 내 정의 참조
```

**이유:** 대화에서 한국어로 호출하므로 EN본에서도 매칭 필수.

### 2. 확신도 형식

```
"확신도: [수치]([근거])"
```

**이유:** 출력 형식이므로 언어 무관 고정.

### 3. 고유 개념명

```
절대자(absolute-executor), 맹점보완(blind-spot补完),
모래시계(inv_pyramid→spine→pyramid), 자율진행(autonomous proceed)
```

**이유:** KR(EN) 병기 형태로 양쪽 참조.

### 4. 금지 표현 리스트

```
"죄송" / "좋은 질문" / "도움이 되셨길" / "궁금한 점" / 인사 / 감사
```

**이유:** 한국어 응답에서 매칭해야 하므로 원문 유지.

### 5. 내부어휘 치환 MAP

```
MAP:
  "spine" → "핵심 메시지 정합성"
  "MECE" → (삭제, 결과만 서술)
  "트리아지" → "우선순위 선별"
  "갭렌즈" → (삭제)
  "축별 분석" → "고객·시장·경쟁 분석"
  "6축 프레임워크 분석" → 각 축 별도 헤딩
  "외화단독표기" → KRW병기(₩1,383/USD 기준)
```

**이유:** 한국어 산출물 내 탐지 대상이므로 원문 유지.

## 영문 변환 대상

위 5개 범주에 해당하지 않는 **규칙 서술부**만 영문 변환.

- DSL 키워드(IF/ENUM/GATE/PIPE 등) = 이미 영문 → 유지
- 기호 연산자(→ ✗ ≥ ≤ × ∝) = 유지
- 나머지 한국어 설명문 = 영문 변환
